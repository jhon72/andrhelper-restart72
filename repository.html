ДРАЙВЕРА

ADB Drivers -

Драйвер необходимый для прошивки устройств через загрузчик FASTBOOT, содержимое архива распаковывается и кладётся в корень диска C в папку с прошивкой

OEM Drivers -

Драйвера для различных устройств на базе OS Android

Google Drivers -

Драйвера для устройств компании Google

MTK Drivers -

Драйвера для смартфонов основанных на чипсетах Mediatek (MTK)

MTK (VCOM) Drivers -

драйвера для смартфонов основанных на чипсетах Mediatek (MTK) (Универс.)

Intel Drivers -

Драйвера для смартфонов основанных на чипсетах Intel

ПО ДЛЯ ПРОШИВКИ УСТРОЙСТВ ANDROID

CMD -

Интерпретатор командных строк, встроенный по умолчанию в ОС Windows, позволяет прошивать устройства при наличии ADB драйвера и драйвера устройства, через режим загрузчика (FASTBOOT) различными командами, начинающиеся с fastboot. Большинство устройств можно прошить через данный интерпретатор командных строк, а некоторые устройства и вовсе можно перешить без использования компьютера.

ODIN -

ПО для прошивки устройств SAMSUNG

SP Flash Tool -

ПО для прошивки устройств на базе чипсетов Mediatek (MTK)

KDZ Updater -

ПО для прошивки некторых устройств LG, у котрых расширение прошивки .kdz

КАСТОМНЫЕ МЕНЮ ВОССТАНОВЛЕНИЯ (СТОКИ ИНДИВИДУАЛЬНЫ)

TWRP -

Кастомное меню восстановления Team Win Recovery Project для различных устройств

CWM -

Кастомное меню восстановления Clockwork mod recovery через ПО ROM Manager для различных устройств на базе ОS Android

КАСТОМНЫЕ ЯДРА

ElementalX -

Кастомное ядро для ограниченного количества устройств Android

Kernel Manager -

Менеджер кастомных ядер
